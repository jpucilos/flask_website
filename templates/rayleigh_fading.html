Rayleigh Fading Simulation

This project was originally a homework assignment for CPE 441, a class I took on Wireless systems. Rayleigh fading is a statistical model for the effect of a propagation environment on a radio signal, such as that used by wireless devices. This script allows you to input values for parameters such as movement speed, sample rate, wavelength, and # of paths and receive an appropriate model of the resultant signal.
